## Getting Started

### Installation

Clone the project, navigate to the `/simple-api` directory, and install dependencies with:

```sh
npm install
```

### Usage

1. Start dev server

```sh
npm run dev
```

2. Open browser or Postman to: `http://localhost:3000/ping`

You should get `{ message: "pong" }` in response
